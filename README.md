# 3.API Documentation:

GET /inhabitants

Returns a formatted list of all inhabitants.

GET /inquiries

Returns a formatted list of all inquiries.

POST /inquiries

Creates a new inquiry

PATCH /inquiries/:id

Changes inquiry status provided with the id

DELETE /inquiries/:id

Deletes an inquiry provided with the id
